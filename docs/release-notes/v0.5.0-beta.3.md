## 🏔️ v0.5.0-beta.3 — "The Lhotse Face"

The ropes are fixed, Camp II is behind us, and the team has pushed into the steepest section of the climb. The Lhotse Face is where the mountain stops being polite — everything gets more vertical, more demanding, and every piece of gear has to perform flawlessly. This release overhauls the entire dialog system with native hybrid modals, adds physical iOS device support, and polishes every page to a uniform, squircle-cornered shine. 132 files touched. No shortcuts on this face.

### 🆕 New Features

- **Native Hybrid Modal System** — Every dialog in the app now uses a hybrid architecture: native macOS/Windows/Linux title bars and buttons with a Blazor WebView for content. The result feels like a real desktop app, not a web page in a trenchcoat.
- **Wizard Modal Infrastructure** — New `WizardFormPage<T>` base class powers multi-step wizards with native Back/Cancel/Next/Submit buttons and a `HybridFormBridge` for real-time state sync between native and Blazor layers.
- **CI Secrets Wizard Overhaul** — The CI Secrets Wizard is now a proper native modal. The monolithic "Resources" step has been split into four focused steps: Bundle ID → Certificate → Provisioning Profile → Configuration, each with search filters.
- **Physical iOS Device Inspector** — Full inspector support for physical iOS devices: app management, log streaming via `idevicesyslog` (or `pymobiledevice3` for iOS 17+), app container downloads, and real-time syslog with channel-backed buffering. The mountain's got cell service now.
- **Repository Picker for Secret Publishing** — Publishing secrets no longer blindly picks the first repo. When multiple repositories are available, an action sheet lets you choose the right destination.

### ✨ Improvements

- **Squircle Everything** — Cards, search boxes, selection summaries, warning banners, modals, and buttons all use consistent squircle corner radii. Borders removed globally. The UI is smoother than freshly groomed snow.
- **Connected Card Lists** — Doctor, SDK packages, workload manifests, and push history now use connected card layouts with clean separator lines instead of individually bordered boxes.
- **Unified Search Box Styling** — Search inputs consolidated into a single global component in `app.css`, replacing duplicated scoped styles across 7+ components.
- **Dedicated Secondary Button Palette** — New `--btn-secondary-bg` and `--btn-secondary-hover` CSS variables give secondary buttons their own identity, distinct from card backgrounds.
- **Settings as Hybrid Dialog** — Settings opens as a native modal from the toolbar, with proper content padding, no duplicate titles, and working Cancel/Save buttons.
- **Auto-Scroll Multi-Operation Modal** — Batch operation modals now automatically scroll to keep the latest progress visible. Font Awesome icons replace emojis.
- **Certificate Sync Badges in Create Profile** — The Create Profile wizard now shows sync status badges on each certificate, matching the Edit Profile dialog.
- **Device Search in Edit Profile** — Added a search filter to the device list in the Edit Profile dialog.

### 🐛 Bug Fixes

- **Dark Mode Flash on Launch** — WKWebView is now hidden natively until Blazor content renders, then fades in over 300ms. No more dark rectangle greeting you at the door.
- **Theme Separator Glitch** — Dialog separators now respond to both system and manual theme changes via `IThemeService` directly, fixing macOS AppKit's silent treatment of `RequestedThemeChanged`.
- **Dark Mode Profile Dialog Colors** — Fixed hover and selected states using undefined `--hover-bg` variable; now correctly uses `--bg-hover`.
- **macOS Export Silently Failing** — `PickSaveFileAsync` had no macOS implementation, falling through to `null`. Added `NSSavePanel` support so settings export actually saves.
- **macOS File Picker Crash** — Added `DevicePlatform.macOS` to `FilePickerFileType` dictionary, preventing `PlatformNotSupportedException`.
- **Physical Device UDID Resolution** — `idevicesyslog` requires the hardware UDID, but the inspector was passing the CoreDevice identifier. Now resolves the correct ID before starting the log stream.
- **Wizard Identity Not Selected** — The CI Secrets Wizard now auto-selects the first Apple identity when opened before visiting Apple pages, preventing empty Bundle ID/Certificate/Profile lists.
- **Crash on Failed Operation Modal** — Fixed a crash when an operation modal reported failure with no result.

### 🔧 Infrastructure

- **Linux Native Packaging Split** — CI now runs separate x64 and arm64 Linux build jobs, with correct artifact paths and a 512×512 icon for Flatpak.
- **GTK4 0.5.0 Upgrade** — Linux builds upgraded to GTK4 platform packages 0.5.0 with native file/folder/save dialogs, debounced toolbar rebuilds, and modal sizing for all 19 dialog types.
- **Windows Modal Polish** — Titlebar controls suppressed during modals, missing SF Symbol icons mapped to Fluent equivalents, and hybrid form dialog backgrounds fixed for dark mode.
- **MauiDevFlow 0.12.1** — Updated CLI and NuGet packages across all three platform projects.
- **Preview SDK Support** — `global.json` updated to allow preview .NET SDK versions.
